# add-space-between-latin-and-cjk
# 用途

在所有中文和英文之间增加空格，增进阅读效果。

## 使用方法

```
python add-space-between-latin-and-cjk.py [folder]
```


### 注意事项

- 可处理选定文件夹及所有子文件夹下的文档。
- 默认只修改`.md`后缀的markdown文档。
- 修改后覆盖原文件，使用前请注意备份。
